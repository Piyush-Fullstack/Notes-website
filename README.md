# Notes Area  
A real-time collaborative notes app built with Socket.io, Node.js and a simple frontend UI. Users can create, edit and update notes, and all changes are synced across connected clients in real time.

## Features
- Create and edit notes  
- Real-time syncing using Socket.io  
- Auto save to database  
- Simple and clean UI  
- REST + WebSocket hybrid architecture
